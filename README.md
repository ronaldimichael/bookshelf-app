# 🚀 Bookshelf App — Smart Reading Management

## 📌 Overview
Bookshelf App is a modern, responsive web application designed to help users efficiently manage their personal reading list. It provides a seamless experience for organizing books into “Completed” and “In Progress” categories, with real-time updates and persistent storage using browser-based technology.

This project reflects strong fundamentals in frontend engineering, focusing on clean UI, efficient state handling, and practical implementation of browser APIs.
